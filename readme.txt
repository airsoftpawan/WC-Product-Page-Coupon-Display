Display coupons below the add to cart button on WooCommerce product pages.

== Description ==
This plugin displays available coupons below the add to cart button on WooCommerce product pages, helping customers find and use coupons easily.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/wc-product-page-coupon-display` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Visit any product page to see the coupons displayed below the add to cart button.
